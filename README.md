应用提供了强大的数据模糊处理功能，可以保护敏感信息的隐私：

- 🔢 数字模糊：生成与原始数字量级相似的随机数字
- 📝 文本模糊：根据文本特征（中英文）生成自然的替代文本
- 📧 邮箱模糊：保留域名，随机化用户名部分
- 📱 电话模糊：根据号码格式（手机号、座机号等）生成符合规则的随机号码
- 🏠 地址模糊：根据地址特征生成中文或英文的随机地址
- 🆔 身份证模糊：保留地区编码，随机化其他部分并保持有效性
- 📅 日期时间模糊：生成有效的随机日期时间，保持原始格式

所有模糊处理都保持数据的格式特征和结构，使处理后的数据看起来自然真实。


## 数据安全说明

数据模糊处理功能旨在保护敏感信息，适用于以下场景：

- 📊 数据分析报告中隐藏真实用户信息
- 🧪 测试环境使用真实数据结构但模糊化的内容
- 📷 截图或分享时保护个人隐私信息
- 📚 教学演示中使用看起来真实但已匿名化的数据
